# BiomedicalComplex
На даний момент створена модель біомедичного комплексу на базі платформ Arduino UNO та NodeMCU, яка може вимірювати опір шкіри, вологість та температуру дихання, а також передавати їх клієнтам що знаходяться в локальній мережі.
Розархівувати архіви. В них будуть папки з Arduino IDE проектами. У папках наявні програми, які необхідно скомпілювати та можна завантажувати на обладнання.
У файлах по-рядкового опису надана основна інформація про програму та використані бібліотеки та модулі.
Особливістю проекту є необхідність встановлення ESP Spiffs Uploader, ESPboard. Також необхідно встановити бібліотеки для створення асинхронного серверу.
